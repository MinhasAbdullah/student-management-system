# Student Management System (Flask)

A web-based Student Management System built with Flask and SQLAlchemy that allows administrators to manage student records.

## Features

- Admin login authentication
- Add student records
- Update student information
- Delete student records
- Search students
- Upload student photos
- Student statistics dashboard

## Technologies Used

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML
- CSS
